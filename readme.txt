The Electric Sheep is a cyborg mind. It harnesses the collective intelligence of 450,000 computers and people to create abstract art with mathematics and Darwinian evolution. The result is seamless, organic, and infinite. See also (electricsheep.org)[http://electricsheep.org] and <a href="http://scottdraves.com">scottdraves.com</a>.<p>
See the <a href="http://code.google.com/p/electricsheep/w/list">Wiki</a> and <a href="http://code.google.com/p/electricsheep/wiki/FAQ">FAQ</a>.<p>
The rendering engine is a separate project at <a href="http://code.google.com/p/flam3/">FLAM3</a>.
